================================================================================
  üîç HOW IT WORKS
================================================================================

1. User speaks ‚Üí Twilio captures audio
2. Audio streamed via WebSocket
3. Converted to 16kHz PCM
4. Voice Activity Detection (user finished?)
5. NeMo transcribes to text (200-500ms)
6. Added to conversation context
7. Full conversation sent to LLM
8. LLM reasons about:
   - What user wants
   - Which tools to call
   - How to respond naturally
9. LLM calls tools if needed
10. LLM generates natural response
11. ElevenLabs converts to speech
12. Audio streamed back to caller
13. User hears response!

Total latency: 0.5-1.5 seconds
